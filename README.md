# Launcher de Videojuegos
WindowsForm con 4 cuatros videojuegos creados en C#

## Snake 🐍
  Controles
  a - Izquierda
  s - Abajo
  d - Derecha
  w - Abajo
  
## Simon Dice 🤔
  Controles                                  
  Click Izquierdo - Seleccionar
  
## Pegale al topo 🔨
  Controles                                  
  Click Izquierdo - Golpear
  
## Flappy bird 🐤
  Controles:                                    
  Espacio - Volar
  R - Reiniciar
